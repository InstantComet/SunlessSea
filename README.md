# 使用方法：

四个文件夹在"\AppData\LocalLow\Failbetter Games\Sunless Sea"目录进行替换

两个dll文件在"\SunlessSea\Sunless Sea_Data\Managed"目录进行替换

resources.assets文件在"\SunlessSea\Sunless Sea_Data"目录进行替换
