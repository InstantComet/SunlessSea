# 使用方法：

所有文件夹在以下目录进行替换：
```
%AppData%\..\LocalLow\Failbetter Games\Sunless Sea
```


两个dll文件在以下目录进行替换
```
你的steam库路径\steamapps\common\SunlessSea\Sunless Sea_Data\Managed
```
resources.assets文件在以下目录进行替换
```
你的steam库路径\steamapps\common\SunlessSea\Sunless Sea_Data
```
